Linux Command Table

Name Description Example
ls The command lists the files of a directory. sp1a@rpi18: ~ $ ls readme tmp
cd The command changes the working directory or takes the user back to home. sp1a@rpi18: ~ $ cd /bin sp1a@rpi18: /bin $
pwd The command displays the current directory. sp1a@rpi18: ~ $ pwd /home/spal
mkdir The command makes a new directory. sp1a@rpi18: ~ $ mkdir mydir sp1a@rpi18: ~ $ ls mydir read me tmp
cp The command copies files and directories. sp1a@rpi18: ~ $ cp readme readme2 sp1a@rpi18: ~ $ ls mydir readme readme2 tmp
mv The command moves files and directories. sp1a@rpi18: ~ $ mv readme readme3 sp1a@rpi18: ~ $ ls mydir readme2 readme3 tmp
rm The command removes files and directories. sp1a@rpi18: ~ $ rm readme3 sp1a@rpi18: ~ $ ls tmp
cat The command displays a file. sp1a@rpi18: ~ $ cat file1.txt
more The command displays a file page by page. sp1a@rpi18: ~ $ more sample.txt
less The command is a more powerful version of ''more''. sp1a@rpi18: ~ $ less sample.txt
man The command gives you access to a reference manual for commands, programs, and functions. sp1a@rpi18: ~ $ man printf
